# Code w Claude
---
Welcome to my repository for projects and solutions built in collaboration with Claude, Anthropic's AI assistant. This space documents my exploratory journey beyond the classroom — tackling real-world problems with code that is, admittedly, a few levels ahead of my current curriculum.
I am currently undergoing formal retraining as an IT Specialist for Data and Process Analysis, which I started on December 15, 2025 — with zero prior programming experience. The projects in this repository were not assigned to me. They emerged from actual problems I wanted to solve, and Claude helped me solve them. Think of it as pair programming, just with an AI that doesn't judge you for not knowing what a for-loop is yet.
## Objective
The purpose of this repository is to bridge the gap between what I am learning in class and what I actually want to build. By working through real use cases with AI assistance, I develop an intuition for software design, APIs, and automation — even before formally covering these topics in my curriculum.
## Tech Stack & Tools:
* Languages: Python (so far)
* Frameworks & Libraries: Flask, Playwright, Requests, BeautifulSoup
* Hosting: PythonAnywhere, Render.com
* Automation: iOS Shortcuts
* Version Control: Git / GitHub
* AI Assistant: Claude (Anthropic)
